This is a plugin and Web Application built for the X-Plane Flight Simulator.

Despite the name, this version is not only for convinent pausing actions during your flight, it presents more features to enjoy your interaction with X-Plane using your Desktop, Tablet or Mobile.

The xpp4me is composed of two parts:

  • X-Plane Plugin: built in C++, it is the plugin itself, working installed at the X-Plane, communicating with the Web Application.
  • Web Application: built in Angular/Ionic/Javascript/CSS/HTML/Python technology, it is the Web Application interacting with X-Plane Plugin, providing a sort of cockpit to xpp4me

Features

  • A Map (Google Map style) to track your airplane during your flight
  • Airplane avionics information (altitude, vertical speed, ground speed, distance)
  • Draw your Flight Plan at the Map
  • Information of each Waypoint at the Flight Plan
  • Using the Map to configure Pause event (distance in nm), interacting with the Waypoints
  • Pause the X-Plane from your Desktop, Mobile or Mobile
  • Configure notifications (by time or distance) that it's about to happen
  • Configure a pause event by a specific time
  • Configure a pause event based on Min / Max Speed
  • Configure a pause event based on Min / Max Altitude

X-Plane Plugin (Installed at your X-Plane)




01. Configuration of a pause event based on the Radio Navigation DME (Frequencies Nav1/Nav2), using min and max comparisons of Time and/or Distance. It is also possible to enable a pause event using the HSI Alignment(if zeros)

02. Configuration of a pause event when your airplane reaches out a minimum or maximum altitude (FT)

03. Configuration of a pause event when your airplane reaches out a minimum or maximum airspeed (KTS)

04. Pause when reaches the configured time

05. Pause event that uses the airplane GPS equipament, comparing the min and/or max of Distance and/or Time, of the current set Waypoint

06. According to the airplane's earth coordinate (latitude and longitude), if matches, a event pause is triggered.

07. Enable a pause event based on the minimum distance of any of the configured NAVAIDs

08. This is likely the most flexible way to configure a pause event, using the Datarefs of X-Plane. You must fill up with the correct path of a existent Dataref, and at the second inputbox the value for comparison. In case of the value, when any signal is added together, the equal (=) is used, also you can use greater than (>=) and lesser than (<=)

09. Here you can fill up or paste a series of Navigational Waypoints(NAVAIDs - Airports, VORs, Fixes, NDBs), like your Fligth Plan, and than send it to the Web Application (using the Send button,n.o 15)

10. In case your Flight Plan contains AIRWAYS on it (like UN975, UM731), the Plugin will attempt to translate it to the sequence of NAVAIDs that compose the Airway

11. If you use the functionality of the "Translate Flight Plan", previous item explained, you have the chance to bring back your original Flight Plan with all your Airways, removing the inserted translated Navaids

12. Copy to Clipboard your current Flight Plan (Ctrl + C does not work)

13. Paste whatever you have on your Clipboard to the Flight Plan inputbox

14. Clean the Flight Plan inputbox

15. As mentioned at the 09 item, this button sends to all of your connected devices using the Web Application MapPause4Me. This way they receive and paint the underline Flight Plan

16. Save the current state, any information on this window will be stored to load again next time (Flight Plan, Distances min and max, etc.)

17. Exit

18. This is the switch(ON/OFF) for the Transmitter of airplane's data to the MapPause4Me Web Application. Obviously this must be STARTED in order the Web Application works with your X-Plane

MapPause4Me Web Application

The Web Application is pretty self-explanatory, it is not very complex to understand the available features. Right after launch it, it will open the Map where you can interate with your airplane's moviments, after connect to it.

There are three menu options available, click at the toggle menu icon (left upper corner) to access them:

Settings - Configuration of the connection with X-Plane, here you must set your Computer's IP Address, where the X-Plane is running, before trying to connect. Find out your Computers IP veryfing your Network Settings, at Properties.



PauseForMe - This is where you can view, define and modify all the pause events, based on the all the available options. You can set a pause event, for instance, when the Airplane reaches out a specific altitude, and this configuration will be send to the X-Plane's Plugin.



Notifications (Working on it...) - Here you can configure notifications, based on time or distance, that is, you can receive a notification when the airplane is 5 nm from a specific Waypoint, or... 3 minutes before reaches it. Notice! This feature works on iOS and Android versions of the MapPause4Me Web Application, not on Browsers. I am still working on them to a future release (when I have some time available to do it). Meanwhile it is possible to use Mobile/Tablet accesing with the Browser, where this Notification will not work properly.
