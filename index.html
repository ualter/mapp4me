This is a plugin and Web Application built for the X-Plane Flight Simulator.

Despite the name, this version is not only for convinent pausing actions during your flight, it presents more features to enjoy your interaction with X-Plane using your Desktop, Tablet or Mobile.

The xpp4me is composed of two parts:

  • X-Plane Plugin: built in C++, it is the plugin itself, working installed at the X-Plane, communicating with the Web Application.
  • Web Application: built in Angular/Ionic/Javascript/CSS/HTML/Python technology, it is the Web Application interacting with X-Plane Plugin, providing a sort of cockpit to xpp4me

Features

  • A Map (Google Map style) to track your airplane position, during your flight
  • Airplane avionics information (altitude, vertical speed, ground speed, distance)
  • Draw your Flight Plan at the Map
  • Information of each Waypoint on Flight Plan
  • Using the Map to configure Pause event (distance in nm), interacting with the Waypoints
  • Pause the X-Plane from your Desktop, Mobile or Mobile
  • Configure notifications of pause events (by time or distance) that it's about to be triggered
  • Configure a pause event by a specific time
  • Configure a pause event based on Min / Max Speed
  • Configure a pause event based on Min / Max Altitude

Installation

Install Python. Download the Plugin and unpackaged it at your X-Plane installation folder, at the plugin's path. Download the Web Application, unpackaged it at a folder of your choice.
